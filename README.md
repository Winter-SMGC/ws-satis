# ws-satis

Registro privado de pacotes Composer da organização Winter-SMGC. Este repositório atua como o hub central para hospedar e distribuir plugins e temas internos via [Satis](https://github.com/composer/satis) e GitHub Pages.

**URL do Registro:** `https://winter-smgc.github.io/ws-satis/packages.json`

## Sobre

O `ws-satis` indexa e disponibiliza os repositórios desenvolvidos internamente. A configuração deste registro já está embutida no boilerplate `winter-smgc/core`, permitindo que os projetos instanciados instalem pacotes nativamente, sem configurações adicionais no `composer.json` do cliente.
